# Subcatica — История кошек

Дипломный проект, представляющий собой интерактивное путешествие по истории эволюции кошек: от современных домашних любимцев до их древних диких предков.

## О проекте
Сайт реализован в формате **Longread** с использованием эффектов параллакса. По мере скролла вниз пользователь «погружается» в историю, встречая различных представителей семейства кошачьих, которые оживают благодаря анимации.

## Технологии
В проекте использованы современные инструменты веб-разработки:
* **HTML5/CSS3** — структура и стилизация.
* **JavaScript (Vanilla)** — логика приложения.
* **GSAP (GreenSock Animation Platform)** — основная библиотека для плавной анимации.
* **ScrollTrigger** — плагин для привязки анимаций к прокрутке страницы.

## Структура папок
* `/assets/img` — изображения кошек, фоны и иконки.
* `/css` — стили проекта (включая настройки параллакса и скрытие скроллбара).
* `/js` — скрипты и инициализация анимаций.
* `index.html` — главная страница проекта.

## Как запустить
1. Склонируйте репозиторий или скачайте архив с проектом.
2. Откройте файл `index.html` в любом современном браузере.

## Особенности
* Интерактивный параллакс вырезанных объектов.
* Кастомный индикатор прогресса (Progress Bar) в верхней части экрана.
* Минималистичный дизайн без стандартного скроллбара браузера для полного погружения.
